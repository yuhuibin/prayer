# 《心经》手抄与学习记录

这是一个用于每日手抄、校对与学习《般若波罗蜜多心经》的个人仓库。

## 项目内容

这个仓库主要记录两类内容：

- 每日手抄后的《心经》文本
- 与当天抄写对应的学习笔记和练习提示

每一天的内容会尽量保存在同一个文件里，方便回看。

## 文件命名

每日文件使用 `心经-YYYYMMDD.md` 的格式，例如：

- `心经-20260421.md`
- `心经-20260422.md`
- `心经-20260423.md`

## 每日流程

当前采用固定节奏：

1. 先查看《心经》原文
2. 完成当天手抄
3. 对照常见流通版逐字校对
4. 补充一点当天的学习内容
5. 将学习内容追加到当天文件底部
6. 提交并推送到 GitHub

## 文本说明

仓库中的《心经》正文以常见流通版为基础，主要用于个人抄写与学习记录。

## 仓库结构

```text
.
├── README.md
├── 心经-20260421.md
├── 心经-20260422.md
├── 心经-20260423.md
└── .claude/
```

## 目标

- 按天积累手抄内容
- 持续补充简明的学习笔记
- 按经文顺序逐步完成整部《心经》的学习整理
